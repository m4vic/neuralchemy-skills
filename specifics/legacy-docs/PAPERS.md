# Research Reading List (Neuralchemy)

Focused list for LLM security, jailbreaks, prompt injection, and evaluation.

## Attacks / Prompt Optimization
- Universal and Transferable Adversarial Attacks on Aligned Language Models (GCG): https://arxiv.org/abs/2307.15043
- AutoDAN: Interpretable Gradient-Based Adversarial Attacks on Large Language Models: https://arxiv.org/abs/2310.15140

## Defenses (practical)
- SmoothLLM: Defending Large Language Models Against Jailbreaking Attacks: https://arxiv.org/abs/2310.03684

## LLM-as-Judge / Evaluation
- Judging LLM-as-a-Judge with MT-Bench and Chatbot Arena: https://arxiv.org/abs/2306.05685
- Large Language Models are not Fair Evaluators: https://arxiv.org/abs/2305.17926

## Prompt Injection in RAG / Agents (useful next)
- (Ab)using Images and Sounds for Indirect Instruction Injection in Multi-Modal LLMs: https://arxiv.org/abs/2307.10490
- Backdoored Retrievers for Prompt Injection Attacks on Retrieval Augmented Generation: https://arxiv.org/abs/2410.14479
- Securing AI Agents Against Prompt Injection Attacks: https://arxiv.org/abs/2511.15759
