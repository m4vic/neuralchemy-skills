# Contributing Guide (Neuralchemy Academy)

Thanks for improving the Academy.

## What We Accept
- Fix broken links, typos, and formatting
- Add high-quality resources (with a 1-line "why it matters")
- Improve learning paths (prereqs -> milestones -> projects)
- Add projects/challenges with a clear rubric
- Add AI security notes with reproducible demos + mitigations

## Where to Put Things
- Track resources: inside the relevant folder (`Fundamentals/`, `MachineLearning/`, `DeepLearning/`, `GenerativeAI/`, `Tools/`, `SystemDesign/`)
- Projects/challenges: `Projects/README.md` (and optionally `Projects/<project-name>/`)
- AI security: `Ai-security/`
- Papers/reading list: update `PAPERS.md` (and optionally add PDFs under `papers/` if you have redistribution rights)
- Labs/experiments: `labs/`

## PR Checklist
- Keep changes scoped (one track / one project / one cleanup)
- Verify links you add actually work
- Prefer consistent headings + simple Markdown
- Avoid huge link dumps without structure

## Style
- Use clear titles and short sections
- Prefer plain ASCII punctuation (avoid copy/paste encoding artifacts)
