# Machine Learning (Core)

Machine Learning is the foundation for deep learning and for building evaluation instincts (metrics, leakage, generalization) that matter in AI security work.

## Prerequisites
- Fundamentals: `../Fundamentals/README.md`

## What you should be able to do
- Train and evaluate baseline models (regression/classification)
- Do correct splits (train/val/test), avoid leakage
- Choose metrics (precision/recall, ROC-AUC, calibration when needed)
- Debug overfitting/underfitting and data issues

## Core Topics
- Supervised learning: linear/logistic regression, trees, ensembles
- Unsupervised learning: clustering, PCA (basic intuition)
- Feature engineering and baselines
- Model evaluation and error analysis

## High-signal Resources
- StatQuest (intuitive ML explanations): https://www.youtube.com/@statquest
- Scikit-learn docs (pipelines, metrics): https://scikit-learn.org/stable/

## Projects (proof of skill)
- Tabular classification with leakage checks + calibration report
- Imbalanced classification (fraud/spam) with proper metrics
- Time-based split project (predict next-week outcome, no lookahead)
