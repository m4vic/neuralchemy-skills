# Fundamentals (Neuralchemy Core)

This track covers the minimum you need for ML, deep learning (PyTorch), transformers/LLMs, and AI security work.

## 1) Python
You should be comfortable with:
- Functions, classes, modules, and packages
- Errors/exceptions and debugging
- File I/O and basic scripting
- Notebooks vs scripts (when to use each)

Libraries to know early:
- NumPy
- Pandas
- Matplotlib (or a basic plotting stack)

## 2) Math (enough to not get stuck)
Prioritize intuition + ability to compute:
- Linear algebra: vectors, matrices, dot products, eigen intuition
- Calculus: derivatives, chain rule, gradients
- Probability & stats: distributions, expectation/variance, conditional probability
- Optimization: what loss functions mean, overfitting vs underfitting

Recommended resources:
- Khan Academy (stats/probability): https://www.khanacademy.org/math/statistics-probability
- 3Blue1Brown (linear algebra): https://www.3blue1brown.com/topics/linear-algebra
- 3Blue1Brown (calculus): https://www.3blue1brown.com/topics/calculus

## 3) Data Basics (bridge to ML)
You should be able to:
- Load/clean tabular data
- Do basic EDA (distributions, missing values, outliers)
- Split data correctly (avoid leakage)
- Pick the right metric (accuracy is not enough)

## 4) Core Habits (non-negotiable)
- Reproducibility: fixed seeds, saved configs, tracked results
- Clear experiments: one change at a time, write down conclusions
- Read papers/code: summarize the goal, assumptions, and failure modes
