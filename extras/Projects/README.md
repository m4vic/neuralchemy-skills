# Projects (Proof of Skill)

This section is where Neuralchemy Academy turns “learning” into “proof”.

## What belongs here
- Small, finishable projects (1–10 hours)
- Clear inputs/outputs, acceptance criteria, and rubric
- Reproducible instructions (README + minimal setup)

## Starter Project Ideas

### AI Security
- Prompt-injection test suite for a toy RAG app
- Data exfiltration demo + mitigation checklist
- Agent tool-abuse scenarios + guardrails evaluation

### ML / DL
- End-to-end tabular model with proper validation + leakage checks
- Reproduce a small paper result (tiny scale) + writeup

### GenAI / LLMs
- RAG baseline vs. reranker comparison (with eval set)
- Finetuning vs. prompt/RAG tradeoff analysis (small experiment)

## Contributing a Project
Open a PR that adds:
- A short project spec (goal, prerequisites, expected output)
- A rubric (what “good” looks like)
- Repro steps (commands, dependencies)
