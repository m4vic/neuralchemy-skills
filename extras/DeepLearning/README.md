# Deep Learning (PyTorch Core)

This track gets you to the point where you can read, train, and debug neural networks in PyTorch, and then move into transformers/LLMs.

## Prerequisites
- Machine Learning: `../MachineLearning/README.md`
- Fundamentals (math basics): `../Fundamentals/README.md`

## What you should be able to do
- Write a clean training loop (data, model, loss, optimizer, eval)
- Use autograd correctly and debug NaNs/shape issues
- Apply regularization, scheduling, and basic optimization tricks
- Reproduce a result at small scale (with logged configs/metrics)

## Core Topics
- Tensors, broadcasting, autograd
- MLPs, CNN basics, sequence basics (RNNs as historical context)
- Optimization: SGD/Adam, learning rate schedules
- Generalization: regularization, augmentation, early stopping

## High-signal Resources
- PyTorch tutorials: https://pytorch.org/tutorials/
- fast.ai (practical deep learning): https://course.fast.ai/

## Projects (proof of skill)
- Image classifier with strong baselines + error analysis
- Text classifier with a simple encoder + clean eval split
- Training harness template (configs, logging, checkpoints)
