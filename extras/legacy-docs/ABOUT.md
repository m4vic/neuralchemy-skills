# About Neuralchemy

Neuralchemy is a learning + building community focused on:
- ML/LLM engineering fundamentals (PyTorch, transformers, evaluation)
- AI security basics for real systems (prompt injection, jailbreak thinking, safe tool use)
- Strong engineering habits (reproducibility, system design, testing)

This repo is the public Academy: curated links, skill paths, projects, and a research reading list.
