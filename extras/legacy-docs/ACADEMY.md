# Neuralchemy Academy (Core Path)

If you don't want to learn everything, follow this core path:

1) Fundamentals (Python + math + data basics)
2) Machine Learning (classical ML + evaluation)
3) Deep Learning (PyTorch + core architectures)
4) Transformers/LLMs (attention, finetuning, RAG basics)
5) LLM security basics (prompt injection, jailbreak mindset, evaluation)
6) Tooling + systems (workflows, experiments, system design)
